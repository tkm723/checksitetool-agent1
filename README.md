# Web検品ツール - チーム共有ガイド

## 初回セットアップ（1回だけ）

### 1. Pythonのインストール
https://www.python.org/downloads/ から最新版をダウンロードしてインストール。

> ⚠️ インストール時に **「Add Python to PATH」** にチェックを入れてください。

### 2. Googleドライブの同期確認
このフォルダ（`web-inspection`）がPCに同期されていることを確認してください。

---

## 毎回の使い方

### ① `start.bat` をダブルクリック
フォルダ内の `start.bat` をダブルクリックするだけです。

- 初回は自動でライブラリをインストールします（数分かかる場合があります）
- ブラウザが自動で開きます（`http://localhost:8501`）

### ② 検品の実行
1. 左サイドバーに **既存サイトのURL** と **検品対象のURL** を入力
2. ワイヤーフレームPPTX（あれば）をアップロード
3. **「検品開始」** ボタンをクリック
4. 結果が出たら **「Excelダウンロード」** でファイルを保存

### ③ 結果の共有
ダウンロードしたExcelファイルを `results/` フォルダに保存すると、チーム全員がGoogleドライブ上で確認できます。

---

## LLM（AI）モードについて

サイドバーの「🤖 LLMによるセマンティックチェック」をONにすると、以下が追加でチェックされます：

- 医療広告ガイドライン違反・グレーゾーン表現
- 誤字脱字・テキスト品質
- 無料提供表現の判定
- 症例ページ・未承認薬の記載

使用にはAnthropicのAPIキーが必要です。`.streamlit/secrets.toml` を作成してください：

```
ANTHROPIC_API_KEY = "sk-ant-xxxxxxxxxxxxxxxx"
```

> APIキーはチームで1つ共有できます。使用量に応じて課金されます（目安：1サイト$0.2〜1）。

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| `start.bat` をダブルクリックしても何も起きない | 右クリック→「管理者として実行」を試す |
| ブラウザが開かない | `http://localhost:8501` を手動でブラウザに入力 |
| `Python が見つかりません` と出る | Pythonを再インストール（PATHチェックを忘れずに） |
| ポート8501が使用中 | 別のStreamlitを閉じてから再起動 |
